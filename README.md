# Supplementary Materials for the Systematic Review

## Title
Gamification as an Information Technology Tool for Enhancing Cognitive Skills and Critical Thinking: A Systematic Review

## Description
This repository contains the supporting materials underlying the systematic literature review conducted in accordance with PRISMA 2020 guidelines.

## Contents
- PRISMA_2020_Checklist.pdf
- PRISMA_Flow_Diagram.pdf
- Dataset_100_Studies.xlsx / .csv
- Search_Strategy.txt
- Inclusion_Exclusion_Criteria.txt

## Methodological Transparency
All materials necessary to verify the review process are provided, including:
- Full database search strategies
- Screening and inclusion criteria
- Final dataset of included studies (n = 100)

## Data Availability
No primary human data were collected. All data consist of bibliographic records and extracted information from published studies.
